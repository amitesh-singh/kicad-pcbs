# Plugins

- KicadStepup
* install flatpak 
* preferences -> kicadstepup -> set the KISYS3DMOD to /usr/share/kicad/3dmodels/
* Add following lines into ~/.profile for GUI apps

```
export KICAD9_3DMODEL_DIR="/usr/share/kicad/3dmodels"
export KISYS3DMOD="/usr/share/kicad/3dmodels"
```
