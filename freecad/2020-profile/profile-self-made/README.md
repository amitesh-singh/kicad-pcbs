# How to make a 2020 or any other profile

Go to mcmaster.com and search for the desired model e.g. 2020
and download the step file

In freeCAD, load the step file and copy it into a new body and apply the Binder
on the face of the profile.
use the binder to create a custom size profile

Refer to `profile-made-from-stepfile.FCStd`
